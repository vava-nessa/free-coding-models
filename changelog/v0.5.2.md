# Changelog v0.5.2 - 2026-06-01

### Added
- Added `minimax-m3-free` to the OpenCode Zen provider — **MiniMax M3 Free**, the new flagship free coding model on Zen, with **1M token context** (10x the other Zen free models) and MiniMax Sparse Attention (MSA) for fast long-context inference.
- Added the MiniMax M3 model license row to the README's license breakdown table, covering M2, M2.5, and M3 under the same permissive royalty-free terms.

### Changed
- Updated the OpenCode Zen section of the README to list 5 free models (was 4) and bumped the catalog badge to `154+ models` (was `153+`).
- Updated the README hero to reference **MiniMax M3** as a headline supported model (was M2).

### Why
MiniMax M3 is the latest frontier open-weight coding model from MiniMax, with MiniMax Sparse Attention (MSA) enabling 1M token context, native multimodality, and 9.7x faster prefilling / 15.6x faster decoding at million-token contexts. Free on OpenCode Zen until further notice. SWE-Bench Pro score: 59.0% (officially reported on the M3 launch blog).
